# portfolio
# 👨‍💻 Personal Portfolio App

A sleek, professional Android application designed to showcase a developer's skills, projects, and contact information. Built natively in Java, this app serves as an interactive digital resume.

## ✨ Features

- **Home Profile**: A welcoming introduction screen highlighting your primary role and a professional headshot.
- **Skills Showcase**: A dedicated section displaying a comprehensive list of your technical skills and proficiencies.
- **Projects Gallery**: An interactive portfolio of your past work and projects, complete with descriptions and visual thumbnails.
- **Contact Info**: An easy-to-use interface allowing potential clients or employers to reach out directly.
- **Intuitive Navigation**: Seamless transitions between the Home, Skills, Projects, and Contact sections using modern Android fragment navigation patterns.

## 🛠️ Tech Stack

- **Language**: Java
- **Platform**: Android
- **UI Toolkit**: XML Layouts, Material Design Components
- **Architecture**: Fragment-based navigation (`HomeFragment`, `SkillsFragment`, `ProjectsFragment`, `ContactFragment`)
- **Key Components**: `RecyclerView` (for dynamic Skills and Projects lists), `CardView`, and `CircleImageView`.

## 🚀 Getting Started

### Prerequisites
- Android Studio (latest version recommended)
- Android SDK

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/portfolio.git
   ```
2. Open the project in **Android Studio**.
3. Update the static data within the `ProjectsFragment` and `SkillsFragment` to reflect your own resume.
4. Let Gradle sync and build the project to resolve all dependencies.
5. Run the app on an Android emulator or a physical device.
